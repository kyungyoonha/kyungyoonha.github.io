  • API key / 서버 주소등 중요한 정보는 깃허브에 올리지 않기 위함
  • 내 로컬에만 저장되기 때문에 안전
  • 개발자 도구로봐도 변수 처리로 되서 보인다.

  • .env 파일 생성

  • 아래와 같이 사용한다.
  • REACT_AP_<변수명>
